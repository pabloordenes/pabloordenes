## 🚀 Acerca de mí

Soy un desarrollador de software con experiencia práctica en stacks tecnológicos completos, desde el **backend con Java (Spring Boot) y Docker** hasta el **desarrollo móvil híbrido con Ionic/Angular** y **frontend con React**.

Mi enfoque es la aplicación de arquitectura limpia (DDD, MVC) y patrones de seguridad (JWT) para construir soluciones robustas que resuelven problemas reales.

- 🎓 Actualmente en Ingeniería en Informática en INACAP.

- 🛠️ Buscando mi práctica profesional para el verano 2025.
